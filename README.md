# 🌍 CommunityPulse

Smart Community Needs & Volunteer Matching System

---

## 📌 Description
CommunityPulse is a web-based platform that centralizes scattered community data and helps identify urgent needs. It connects volunteers with the right tasks in real time for faster and more efficient response.

---

## 🚀 Features
- Data collection (needs input)
- Need prioritization (high/medium/low)
- Live map visualization
- Volunteer registration
- Smart task matching
- Task tracking

---

## 🛠️ Technologies
- HTML, CSS, JavaScript
- Firebase Firestore
- Leaflet.js
- OpenStreetMap API

---

## ⚙️ How It Works
1. User adds a need (type, city, urgency)
2. System stores and categorizes it
3. Need appears on live map
4. Volunteer registers
5. System matches volunteer to need
6. Task is completed and updated
